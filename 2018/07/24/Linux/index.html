title: Linux
date: 2018-07-24 15:08:53

tags:


Linux 严格区分大小写
Linux 所有内容以文件形式保存 而且文件类型与文件后缀无关

ls： 列出当前/相关目录下的所有文件和目录
-a 显示所有文件
-l 详细信息显示
-d 查看目录属性

颜色类型

mkdir: 创建目录
-p 递归创建
-m 自定义创建目录的权限 eg：mkdir -m 775 xxx
-v 显示创建目录的详细信息

pwd: 显示当前目录

rmdir: 删除空目录
-v 显示执行过程
-p 若自父母删除后父目录为空则一并删除

rm: 删除文件
-r 删除目录
-f 强行执行删除 不询问
-i 删除提示
-v 详细显示进行步骤

cp: 将源文件复制到目标文件，或将多个源文件复制到目标目录中
-r -R 递归复制该目录及其子目录内容
-p 连同档案属性一起复制过去
-f 不询问强行复制
-s 生成快捷方式
-a 将档案所有特性一起复制

mv: 剪切文件，改名
-b 覆盖之前的备份
-f 如存在不询问而强制覆盖
-i 存在则询问是否覆盖
-u 较新才覆盖
-t 将多个源文件移动到统一目录下，目录参数在前，文件参数在后

scp: 在linux服务器之间复制文件和目录
-v 详细显示输出具体情况
-r 递归复制整个目录

cd: 改变当前目录 </!– cd后面有空格 –>

  • 返回上层目录
    .. 返回上层目录
    回车 返回主目录
    / 根目录

echo: 显示内容
-n 输出后不换行
-e 遇到转义字符特殊处理

touch: 创建空文件
eg touch “ic ab” 一个文件
touch ic ab 两个文件

cat： 显示文件内容
-n 显示行号

tac: 反向显示内容

more: 分页显示内容
空格或f 翻页
enter 换行
q或Q 退出
+n 从第n行开始显示
-n 每次查看n行数据
+/String 搜索String字符串位置，从其前两行开始查看
-c 清屏再显示
-p 换页时清屏

less: 分页显示内容
/加关键词 搜索 按n至下一个

head: 显示文件前几行
-n 指定行数

tail: 显示文件末尾几行
-n 指定函数
-f 动态显示文件末尾内容

ln: 生成链接文件
-s 创建软链接
不加s 创建硬链接

硬链接特征

  • 拷贝cp-p加同步更新
  • i节点与源文件相同
  • 不能跨分区

chomd: 更改权限
chomd {ugoa}{+-=}{rwx} 文件或目录
eg chomd u+x 文件名
用数字表示 r=4 w=2 x=1
eg chomd 640 文件名
-R 递归修改

文件
r 可以查看文件内容（cat,more,head,tail,less) 可以列出目录中的文件(ls)
w 可以修改文件内容（vim） 可以在目录中创建，删除文件（touch,mkdir,rmdir,rm)
x 可以这行文件（script command） 可以进入目录（cd）

chown: 改变文件或目录的所有者
chowm [用户] [文件或目录]

charp: 权限管理命令
改变文件或目录的所属组

umask: 显示，设置文件的缺省权限
-s 以rwx形式显示新建文件缺省权限
默认目录755 文件644

find: 文件搜索
find [搜索范围] [匹配条件]
eg find /etc -name init
init 模糊搜索
init??? ？代表一个字符

-iname 不区分大小写

find / -size +204800
在根目录下查找大于100mb的文件
+n 大于 -n小于 n 等于

find /home -user shenchao
在根目录下查找所有者为shenchao的文件
-gourp 所属组…

find /etc -cmin -5
在etc 下寻找5分钟内被修改过属性的文件和目录
-amin 访问时间
-cmin 文件属性
-mmin 文件内容

find /etc -size +163840 -a -size 204800
在etc 下寻找大于80mb小于100mb的文件
-a 两个条件同时满足
-o 两个条件满足一个就行了

find /etc -name inittab -exec ls -l { } \;
在etc下寻找inittab文件显示其详细信息
-exec或-ok 命令 { } \; 对搜索结果执行操作
-ok会询问确认
find -inum 31531 -exec rm { } \;

locate: 在文件资料库中查找文件（快速查找）
locate 文件名
没有在文件资料库中的文件是找不到的
updatedb 更新资料库（不更新tmp中的文件）
locate -i 文件名 不区分大小写

which: 搜索命令所在目录以及别名信息
eg which ls

whereis: 搜索命令所在目录及帮助文件路径
whereis ls

grep: 在文件中搜录字串匹配的行并且输出
-i 不区分大小写
-v 排除指定字串
-c 只输出匹配行地计数
-I 不区分大小写（只适用于单字符）
-l 只显示文件名
-v 显示不包含匹配文本的所有行
grep -iv 关键字 文件地址
grep -iv ^#(#开头） /etc/inittab

man: 获得帮助信息
man 命令或配置文件
man ls 查看ls命令的帮助信息
man services （不用加绝对路径 直接加名称）
查看配置文件services的帮助信息
如果命令与配置文件名字相同 优先查看命令的帮助信息

man 5 passwd 配置文件的帮助信息
1 命令 5 配置文件

whatis: 命令简短信息
whatis 命令名称

apropos: 配置文件简短信息
apropos 配置文件

help: 获得shell内置命令的帮助信息（找不到路径的命令）
help 命令

file: 判断文件类型

zip: 压缩文件或目录 保留原文件
zip [-r] [压缩后文件名] [文件或目录]
-r 压缩目录

unzip: 解压.zip的压缩文件
unzip 压缩文件

bzip2: 压缩文件 （压缩比大）
bzip2 [-k] [文件]
-k 保留原文件

bunzip2: 解压
-k 保留原压缩包
-# 压缩等级
-d 解压
-z 压缩
-c 将压缩过程产生的数据输出到屏幕上
-v 可以显示出原文件/压缩文件的压缩比等信息

gzip: 压缩文件，源文件都不再存在
-d 进行解压缩
-c 将压缩的数据输出到屏幕上
-v 显示原文件/压缩文件的压缩比等信息
-# 压缩等级，-1最快，但压缩比最差，-9最慢，压缩比最好

gunzip: 解压缩文件

tar: 打包目录压缩后文件格式为tar.gz 可以命名这个后缀便于区分
tar [-zcf] [压缩后文件名] [目录]
-c 打包
-v 显示详细信息
-f 指定文件名
-z 打包同时压缩 -j 用bzip2压缩
-x 解包
-z 解压缩 -j 用bunzip2来解压

netstat: 显示网络相关信息
-t tcp协议（3次握手）
-u udp协议 （快）
-l 监听
-r 路由
-n 显示ip地址和端口号
netstat -tlum 查看本机监听的端口
netstat -an 查看本机所有的网络连接
netstat -rn 查看本机的路由表

setup: 配置网络
setup

mount: 挂载设备
mount 设备文件名 挂载点

umount: 卸载
umount 设备文件名

write: 给用户发送信息 以ctrld结束
write 用户名（在线的）

wall: 发广播信息 给所有在线用户发信息
wall [message]

ping: 测试网络连通性 （默认无限次 ctrilC终止）
-c 指定发送次数
ping -c 3 192.168.1.156

ifconfig: 查看和设置网卡信息
ifconfig 查看
ifconfig 网卡名称 ip地址（临时改网络信息）
eg ifconfig etho 192.168.256

last: 列出目前与过去登入系统的用户信息
