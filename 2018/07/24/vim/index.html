vim

Posted on 2018-07-24

在终端输入vimtutor有学习教程

基本操作

操作 作用
vim filename 创建/进入文件
k j h l 分别表示上、下、左、右移动
Esc 退出某个模式

光标移动

操作 作用
e 光标跳转至一个单词的末尾
w 光标跳转至下一个单词开头
$ 光标跳转至行末
i 进入插入模式，在光标位置前插入
I 在光标所在行行首插入
a 进入插入模式，在光标位置后插入
A 在一行后的末尾插入
o 在光标的下一行插入
O 在光标的上一行插入
x 删除光标所在位置
0 移动光标到首行
：n 到第n行
nG 到第n行
gg 到第一行
G 到最后一行

保存退出

操作 作用
：wq 保存并退出
q! Enter 丢弃修改并退出vim

查找指令

操作 作用
/ 搜索内容 搜索内容，n表示往下继续查找，N表示往上查找，反向查找用?代替 /
输入：set ic 忽略大小写
输入：set noic 区分大小写
输入：set (no)hl 是否设置搜索内容高亮
输入：set (no)is 设置查找短语时显示部分匹配
% 查找与光标所在位置相匹配的左、右大括号、中括号、小括号
v 进入可视模式，移动光标会选中内容，可对选中内容进行各种操作
：!+命令 执行外部文件或命令
:set nu 设置行号
：set nonu 取消行号

复制粘贴

操作 作用
y 复制
yy 复制当前行
nyy 复制当前行以下n行
dd 剪切当前行
ndd 剪切当前行以下n行
p，P 粘贴在当前光标所在行下或行上
p 将最后一次删除的内容置入光标所在位置

删除操作

操作 作用
x 删除光标所在位置
nx 删除光标所在处的n个字符
dd 删除光标所在行
ndd 删除n行
dG 删除光标所在行到文件末尾内容
D 删除光标所在处到行尾内容
：n1,n2d 删除指定范围的行
dw 从当前光标当前位置直到下一个单词起始处，不包括它的第一个字符。
de 从当前光标当前位置直到单词末尾，包括最后一个字符。
d$ 从当前光标当前位置直到当前行末。

撤销

操作 作用
u 撤销以前的操作
U 撤销一行中的所有改动
Ctrl R：撤销撤销命令

更改操作

操作 作用
r+字母 将光标所在位置替换成指定字母
ce 要改变文本直到一个单词的末尾，会同时进入插入模式
R 可以连续更改内容

文件定位及操作

操作 作用
Ctrl+G 显示当前编辑文件中当前光标所在行位置以及文件状态信息
:w 文件名 新建一个文件夹保存当前文件
:r 文件名 在光标位置置入指定内容
：r ！命令 导入命令执行结果

替换命令

操作 作用
:s/原内容/替换的内容 该命令只改变光标所在行的第一个匹配项
:s/原内容/替换的内容/g 改变全行的匹配项
:#,#s/old/new/g 其中 #,# 代表的是替换操作的若干行中首尾两行的行号。
:%s/old/new/g 则是替换整个文件中的每个匹配串。
:%s/old/new/gc 会找到整个文件中的每个匹配串，并且对每个匹配串提示是否进行替换。(y表示是，n表示不，q表示退出，l表示离开)
:n1,n2s/old/new/g 在一定范围内替换指定字符串

连续行注释

替换
