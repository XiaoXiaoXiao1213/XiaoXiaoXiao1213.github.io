vim

Posted on 2018-07-24

在终端输入vimtutor有学习教程

###基本操作
|vim filename| 创建/进入文件|
| k j h l|分别表示上、下、左、右移动|
| Esc |退出某个模式|

###光标移动
|e |光标跳转至一个单词的末尾|
|w |光标跳转至下一个单词开头|
|$ |光标跳转至行末|
|i |进入插入模式，在光标位置前插入|
|I|在光标所在行行首插入|
|a|进入插入模式，在光标位置后插入|
|A|在一行后的末尾插入|
|o|在光标的下一行插入|
|O|在光标的上一行插入|
|x|删除光标所在位置|
|0|移动光标到首行|
|：n|到第n行|
|nG|到第n行|
|gg|到第一行|
|G|到最后一行|

###保存退出
|：wq|保存并退出|
|q! Enter |丢弃修改并退出vim|

###查找指令
|/ 搜索内容|搜索内容，n表示往下继续查找，N表示往上查找，反向查找用?代替 /|
|输入：set ic|忽略大小写|
|输入：set noic|区分大小写|
|输入：set (no)hl|是否设置搜索内容高亮|
|输入：set (no)is|设置查找短语时显示部分匹配|
|%|查找与光标所在位置相匹配的左、右大括号、中括号、小括号|
|v|进入可视模式，移动光标会选中内容，可对选中内容进行各种操作|
|：!+命令|执行外部文件或命令|
|:set nu|设置行号|
|：set nonu|取消行号|

###复制粘贴
|y|复制|
|yy|复制当前行|
|nyy|复制当前行以下n行|
|dd|剪切当前行|
|ndd|剪切当前行以下n行|
|p，P|粘贴在当前光标所在行下或行上|
|p|将最后一次删除的内容置入光标所在位置|

###删除操作
|x|删除光标所在位置|
|nx|删除光标所在处的n个字符|
|dd|删除光标所在行|
|ndd|删除n行|
|dG|删除光标所在行到文件末尾内容|
|D|删除光标所在处到行尾内容|
|：n1,n2d|删除指定范围的行|
|dw|从当前光标当前位置直到下一个单词起始处，不包括它的第一个字符。|
|de|从当前光标当前位置直到单词末尾，包括最后一个字符。|
|d$|从当前光标当前位置直到当前行末。|

###撤销
|u|撤销以前的操作|
|U| 撤销一行中的所有改动|
|Ctrl R：撤销撤销命令|

###更改操作
|r+字母|将光标所在位置替换成指定字母|
|ce|要改变文本直到一个单词的末尾，会同时进入插入模式|
|R|可以连续更改内容|

###文件定位及操作

|Ctrl+G|显示当前编辑文件中当前光标所在行位置以及文件状态信息|
|:w 文件名|新建一个文件夹保存当前文件|
|:r 文件名|在光标位置置入指定内容|
|：r ！命令|导入命令执行结果|

###替换命令
|:s/原内容/替换的内容|该命令只改变光标所在行的第一个匹配项|
|:s/原内容/替换的内容/g|改变全行的匹配项|
|:#,#s/old/new/g|其中 #,# 代表的是替换操作的若干行中首尾两行的行号。|
|:%s/old/new/g|则是替换整个文件中的每个匹配串。|
|:%s/old/new/gc|会找到整个文件中的每个匹配串，并且对每个匹配串提示是否进行替换。(y表示是，n表示不，q表示退出，l表示离开)|
|:n1,n2s/old/new/g|在一定范围内替换指定字符串|

###定义快捷键

###连续行注释

###替换

在配置文件写才对所有文件永久有效
用户– /home/username/.vimrc
root – /root/.vimrc

命令时，按 CTRL-D 可以查看可能的补全结果。按可以使用一个补全。

Linux

Posted on 2018-07-24


Linux 严格区分大小写
Linux 所有内容以文件形式保存 而且文件类型与文件后缀无关

ls： 列出当前/相关目录下的所有文件和目录

颜色类型

mkdir: 创建目录

pwd: 显示当前目录

rmdir: 删除空目录

rm: 删除文件

cp: 将源文件复制到目标文件，或将多个源文件复制到目标目录中

mv: 剪切文件，改名

scp: 在linux服务器之间复制文件和目录

cd: 改变当前目（cd后面有空格）

echo: 显示内容

touch: 创建空文件（可同时创建多个文件）

cat： 显示文件内容

tac: 反向显示内容

more: 分页显示内容

less: 分页显示内容（可以往上翻 其他与more相同）

head: 显示文件前几行

tail: 显示文件末尾几行

ln: 生成链接文件

硬链接特征

  • 拷贝cp-p加同步更新
  • i节点与源文件相同
  • 不能跨分区

chomd: 更改权限

文件 目录
r 可以查看文件内容（cat,more,head,tail,less) 可以列出目录中的文件(ls)
w 可以修改文件内容（vim） 可以在目录中创建，删除文件（touch,mkdir,rmdir,rm)
x 可以这行文件（script command） 可以进入目录（cd）

chown: 改变文件或目录的所有者（ 管理员才可以）

charp: 权限管理命令

umask: 显示，设置文件的缺省权限

find: 文件搜索
find [搜索范围] [匹配条件]

locate: 在文件资料库中查找文件（快速查找）
locate 文件名
没有在文件资料库中的文件是找不到的
updatedb 更新资料库（不更新tmp中的文件）
locate -i 文件名 不区分大小写

which: 搜索命令所在目录以及别名信息

whereis: 搜索命令所在目录及帮助文件路径

grep: 在文件中搜录字串匹配的行并且输出

man: 获得帮助信息

whatis: 命令简短信息

apropos: 配置文件简短信息

help: 获得shell内置命令的帮助信息（找不到路径的命令）

file: 判断文件类型

zip: 压缩文件或目录 保留原文件
zip [-r] [压缩后文件名] [文件或目录]

unzip: 解压.zip的压缩文件

bzip2: 压缩文件 （压缩比大）
bzip2 [-k] [文件]

bunzip2: 解压

gzip: 压缩文件，源文件都不再存在

gunzip: 解压缩文件

tar: 打包目录压缩后文件格式为tar.gz 可以命名这个后缀便于区分
tar [-zcf] [压缩后文件名] [目录]

netstat: 显示网络相关信息

setup: 配置网络

mount: 挂载设备
mount 设备文件名 挂载点

umount: 卸载
umount 设备文件名
write: 给用户发送信息 以ctrld结束
write 用户名（在线的）

wall: 发广播信息 给所有在线用户发信息
wall [message]

ping: 测试网络连通性 （默认无限次 ctrilC终止）

ifconfig: 查看和设置网卡信息

last: 列出目前与过去登入系统的用户信息
